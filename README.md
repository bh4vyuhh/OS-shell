# OS-shell

A minimal Unix-like shell implemented in C, built to understand core operating system concepts such as process creation, command execution, pipes, and file redirection.

Features -
Execute system commands (ls, cat, pwd, etc.)
Built-in commands:
cd , help , exit
Input & Output Redirection:
> (write to file)
< (read from file)
Pipes:
| (single pipe support)

Commands
- ls , mkdir , pwd , rmdir , rm , touch , grep , head , tail , cat , exit.

RUN this shell using docker/LinuxVM
